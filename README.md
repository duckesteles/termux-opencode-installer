# termux-opencode-installer

This script compiles, installs and updates OpenCode in your Termux environment. It manages dependencies and automates the packaging process for you.

## Installation

Copy and paste the following command into your terminal to start the installation.

```bash
curl -sL https://raw.githubusercontent.com/duckesteles/termux-opencode-installer/refs/heads/main/install.sh -o install.sh && chmod +x install.sh && ./install.sh
```
